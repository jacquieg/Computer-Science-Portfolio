What Is HTML

Today we will be talking all about HTML.It stands for ""HyperText Markup Language" and it's used to create webpages using all sorts of different tags which make up the structure of a webpage. Using these tags can help you create your headings, paragraphs, lists, links, quotes and make up a webpage like the one you see now. HTML was created by a man named Tim Berners-Lee, which was the same man who also created URL and HTTP. This all happened around the year 1989. While Tim Berners-Lee worked at the CERN, he shared and constantly updated his information, which later led him to build a prototype system named ENQUIRE. In later years, he used all his ideas that he had gained from past years of sharing and communicating with his co-workers to combine hypertext idea and connect it to the TCP and DNS ideas and soon after that process the World Wide Web was created. HTML was written used standard markup language .

What Is CSS

Today we will also talk about CSS.It stands for Cascading Style Sheets and it's used by adding a individual HTML element as the value of the style attribute. In other words, CSS is used for describing the look and formatting of a document written in a markup language. CSS was created in 1994 by One of the authors at CERN. It was officially realeased in December 17, 1996, which was about 18 years ago. CSS was written in markup language

All About Tags

A tag is used to style your content, headings and background using CSS. It can also be used to create paragraps and heading as well as lists using HTML. A tag is composed of the name of the element, for example it can be paragraphs or headings, surrounded by angle brackets. An end tag also has to be added with a slash after the opening angle bracket, and this is only to be able to distinguish it from the start tag. Some examples of tags are:

  • The head tag
  • Paragraph tag
  • Body tag
  • list tag

    • All these tags are either used for HTML or CSS. For example if you're going to use the head tag in HTML your most likely adding a heading to your website, but if your using it for CSS you're going to most likely use it for styling your heading. These are just some of many different tags that you can use to style or create something on your webpage.
